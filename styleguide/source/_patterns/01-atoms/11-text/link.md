### Description
This Pattern shows a link with an optional chevron.

### Status
* Stable as of 5.0.0

### Variant options
* A [chevron](./?p=atoms-link-chevron) can be rendered after it


### Variables
~~~
link {
  href:
    type: string / required
  text:
    type: string / required
  info:
    type: string / optional
  chevron:
    type: boolean
}
~~~
