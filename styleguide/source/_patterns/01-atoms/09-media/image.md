### Description
Displays an image using the image HTML element

### Status
* Stable as of 4.0.0

### Variant options
* [circular image](?p=atoms-image-circular)

### Variables
~~~
image {
  alt:
    type: string / required
  src:
    type: string (url) / required
  srcset:
    type: string / optional
  sizes:
    type: string / optional
  height:
    type: string / optional
  width:
    type: string / optional
  shape:
    type: string / optional
}
~~~
