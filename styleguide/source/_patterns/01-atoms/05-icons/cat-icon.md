### Description
This Pattern shows a small category flag.

### Status
* Stable as of 5.0.0

### Variant options
* This can be rendered in a [smaller](./?p=atoms-cat-icon-small) size


### Variables
~~~
catIcon: {
  icon:
    type: string (path to icon) / required 
  small:
    type: string ("true" || "") / optional
}
~~~
