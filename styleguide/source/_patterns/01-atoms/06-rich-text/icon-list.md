### Description
An `<ul>` element with its child `<li>` elements that is displayed with a custom icon bullet in a Rich text editor.  An optional nested child `<ul>` and `<li>` elements is allowed.

### Status
* Stable as of 5.0.0


### Variables
~~~
iconList [{
  text:
    type: string / required
  sublist (optional) [{
    text:
      type: string / required
  }]
}]
~~~
