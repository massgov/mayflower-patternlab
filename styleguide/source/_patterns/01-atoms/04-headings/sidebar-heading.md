### Description
An `<h3>` element used as a subheading.  Typically used in the Right Rail or in a 50/50 column container.

### Status
* Stable as of 3.0.0

### Variables
~~~
sidebarHeading: {
  title:
    type: string / required
  level:
    type: number / optional
  titleContext:
    type: string / optional
}
~~~
