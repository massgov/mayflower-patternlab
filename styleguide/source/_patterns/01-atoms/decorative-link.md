Description: This pattern shows a stylized `a` tag with an arrow icon.

### Status
* Stable as of 4.0.0

### Pattern Contains
* SVG arrow atom

### Variables
~~~
decorativeLink {
  href:
    type: string (url) / required
  text:
    type: string / required
  info: 
    type: string (adds more description to the link) / optional
  property:
    type: string / optional
}
~~~
