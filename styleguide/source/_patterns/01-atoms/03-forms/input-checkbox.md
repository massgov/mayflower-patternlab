### Description
This is a stylized checkbox with a label and an optional icon.

### Status
* Stable as of 5.0.0

### Variables
~~~
inputCheckbox {
  value:
    type: string / required
  id: 
    type: string / required
  icon:
    type: string (path to icon) / optional
  label:
    type: string / required
  checked:
    type: boolean
}
~~~
