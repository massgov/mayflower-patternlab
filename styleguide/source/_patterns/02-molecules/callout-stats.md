### Description
This is Pattern is used to highlight important stats.

### Status
* Stable as of 5.0.0


### Variant options
* This pattern can be floated [left](./?p=molecules-callout-stats-left) or [right](./?p=molecules-callout-stats-right)


### Variables
~~~
statsCallout: {
  pull:
    type: string ('', 'left', 'right')
  stat:
    type: string / required
  content:
    type: string / required
}
~~~
