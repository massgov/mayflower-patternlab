### Description
This is a set of related links with an optional label that are typically used to navigate to a parent page.

### Status
* Stable as of 5.0.0

### Variables
~~~
headerTags: {
  label:
    type: string / optional
  taxonomyTerms: [{
    href:
      type: string / required
    text:
      type: string / required
  }]
}
~~~
