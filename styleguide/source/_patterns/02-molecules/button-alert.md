### Description
This is a black button designed to show and hide content (text toggles as well)

### Status
* Stable as of 5.0.0

### JavaScript Used
* Emergency Alerts (js/modules/emergencyAlerts.js)

### Variables
~~~
buttonAlert: {
  hideText:
    type: string / required,
  showText:
    type: string / required,
  class:
    type: string / optional,
  text:
    type: string / optional
}
~~~
