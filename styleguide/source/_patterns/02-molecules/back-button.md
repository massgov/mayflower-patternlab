### Description
This Pattern is a button that is designed to take the user Back to the Previous page.

### Status
* Stable as of 5.0.0

### Variables
~~~
backButton: {
  link: {
    href: 
      type: string / required
    text:  
      type: string / required
    info:  
      type: string / optional (for screen readers if text isn't descriptive)
  }
}
~~~
