### Description
This is an example of an Organization page for Mass.gov using a stacked row template

### Status
* Stable as of 5.0.0

### Pattern Contains
* Header
* Footer
* Stacked Row template
* Page Banner (large)
* Page Header (location)
* Stacked Row Section
* Rich Text
* Icon Links
* Action Finder
* Mapped Locations
* Section Three Up
* Link List


### Variables
See Patterns above
~~~
pageBanner: {
  type: pageBanner / required
},

pageHeader: {
  type: pageHeader / required
},

stackedRowSections: [{
  type: array of stackedRowSections / required
}]
~~~
