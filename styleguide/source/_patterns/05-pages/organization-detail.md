### Description
This is an example of an Organization Detail page for Mass.gov using a stacked row template

### Status
* Stable as of 5.0.0

### Pattern Contains
* Header
* Footer
* Stacked Row (template)
* Page Banner - large
* Raw HTML
* Image
* Divider


### Variables
See Patterns above
~~~
pageBanner: {
  type: pageBanner / required
},

stackedRowSections: [{
  type: array of stackedRowSections / required
}]
~~~
