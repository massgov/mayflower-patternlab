### Description
This page [extends](https://twig.symfony.com/doc/2.x/tags/extends.html) the location template to show how it could be used as a Park Location page.

### Status
* Stable as of 5.0.0

### Pattern Contains
* Header
* Footer
* Location (template)
* Labelled Icons
* Contact List
* Callout Grid
* Rich Text
* Google Map
* Image Promo
* Event List
* Suggested Pages
