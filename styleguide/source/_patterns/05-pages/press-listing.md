### Description
This is an example of a Press Listing page [extending](https://twig.symfony.com/doc/2.x/tags/extends.html) the Left Rail template

### Status
* Stable as of 5.0.0

### Pattern Contains
* Header
* Footer
* Left Rail (template)
* Page Header
* Press Filters
* Results Heading
* Press Listing (organism)
* Pagination
