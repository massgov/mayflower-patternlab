### Description
This is a Table of Contents Navigation Pattern.

### Status
* Stable as of 5.0.0

### JavaScript Used
* Sticky Nav and smooth scrolling (js/modules/scrollAnchors.js)

### Variables
~~~
jumpLinks: {
  title: 
    type: string / required
  links: [{
    text: 
      type: string / required,
    href: 
      type: string (url) / required,
    info: 
      type: string / optional
  }]
}
~~~
