### Description
This Pattern shows an overlay of expantable table of contents sections.

### Status
* Stable as of 5.11.0

### Pattern Contains
* Table of contents hierarchy

### Usage Guidelines
*

### Variables
~~~
tableOfContentsOverlay: {
  id: {
    type: text / required
  },
  title: {
    type: coloredHeading / required
  },
  tocSections: [{
    type: table of contents hierarchy / required
  }]
}
~~~
