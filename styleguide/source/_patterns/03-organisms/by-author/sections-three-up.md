### Description
This patten shows a grid of Section Link Patterns.

### Status
* Stable as of 5.0.0

### Pattern Contains
* Comp Heading
* Section Links


### JavaScript Used
* This pattern uses JavaScript for the accordions (js/modules/accordions.js)


### Variables
~~~
sectionThreeUp : {
  compHeading: {
    type: compHeading / optional
  },
  "sections": [{
    type: array of sectionLinks / required
  }]
}
~~~
