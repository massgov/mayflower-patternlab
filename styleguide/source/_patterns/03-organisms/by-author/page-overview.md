### Description
This Pattern is used to explain the reason for the content of a page.

### Status
* Stable as of 5.0.0

### Pattern Contains
* Comp Heading
* Image
* Labelled List
* Rich Text

### Variables
~~~
pageOverview: {
  compHeading: {
    type: compHeading / optional
  },
  image: { 
    type: image / optional
  },
  labelledList: {
    type: labelledList / optional
  },
  richText: {
    type: richText / optional
  }
}
~~~
