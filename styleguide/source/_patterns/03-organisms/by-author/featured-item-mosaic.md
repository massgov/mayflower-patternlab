### Description
A 'mosaic' grid of featured items.

### Status
* Stable as of 5.0.0

### Pattern Contains
* Comp Heading
* Featured Item
* Divider


### Variables
~~~
featuredItemMosaic: {
  compHeading:{
    type: compHeading / optional
  },
  columns:[
    [
      type: featuredItem / required
    ]
  ],
  more:{
    type: link / optional
  }
}
~~~
