### Description
This patten shows a grid of Topical Card Patterns.

### Status
* Stable as of 5.0.0

### Pattern Contains
* Comp Heading
* Topical Card

### JavaScript Used
* This pattern uses JavaScript for the accordions (js/modules/accordions.js)


### Variables
~~~
threeUpCards : {
  compHeading: {
    type: compHeading / optional
  },
  "sections": [{
    type: array of topicalCard / required
  }]
}
~~~
